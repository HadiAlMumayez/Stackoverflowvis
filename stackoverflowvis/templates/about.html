Stack overflow Data Visualization

Project Description

Building a website using Django to view the accurate analyzes and studies we've done on Stack Overflow dataset. Stack Overflow (SO) is the largest Q&A website for developers, providing a huge amount of data,. This web sites often rely upon tags as metadata that assists in the indexing, categorization, and search for particular content with just a few key words. The most obvious benefit stems from the fact that tags help ensure that website data is properly categorized and easily searchable by users. This was the biggest motivation for us to conduct large-scale analytical studies on the Stack Overflow social network using graph theory. The underlying structure of the Stack Overflow network is diverse and complex since users and tags can be related through various questions, answers, and comments. We picked certain relationships between objects that we deemed to likely be the most indicative and constructed the appropriate graphs. Some of the results that we tried to find by analyzing the questions asked in Stack Overflow The development of programming languages over time based on use of their tags since 2008, Finding Similar Tags , How many questions are asked with the number of TopTags in countries and What are the most commonly used tags.
